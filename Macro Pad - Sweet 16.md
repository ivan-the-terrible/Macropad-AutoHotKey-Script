# Macro Pad - Sweet 16

1 - Open w/ VSCode
	- F13
2 - Extract File 
	- F14 - 7zip command
3 - Archive Files 
	- F15 - 7zip command
4 - 
	- F16

5 - Color Picker 
	- Windows + Shift  + c
6 - Screen Ruler 
	- Windows  + Shift + m
7 - Always On Top 
	- Windows + Ctrl + t
8 - Mouse Pointer Crosshairs 
	- Windows + Alt + p

9 - Snipping Tool 
	- Windows + Shift + s
10 - Export to PNG
	- F17 - Magick command
11 - Open w/ Inkscape
	- F18
12 - 
	- F19


13 - Volume Down
	- KC_VOLD
14 - Volume Up
	- KC_VOLU
15 - Mute/Unmute Mic
	- F20
16 - Switch Audio Input
	- F21
